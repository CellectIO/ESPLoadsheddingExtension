Welcome to the ESP Loadshedding extension

We're thrilled to have you on board.

This guide is designed to assist you in navigating through the initial setup process necessary for the functionality of this extension.

Whether you're a seasoned user or just getting started, we're here to ensure your experience is as smooth as possible.

Let's embark on this journey together and unlock the full potential of the ESP Loadshedding extension!

Step 1: Sign up for an ESP API key

Go to the Gumroad Dashboard with the link below to sign up for an ESP API Key.

You can copy the link with the copy button on the right and then paste the link in a new browser tab.

Copy Gumroad API Link

Once you are on the page you can click on the Subscribe button on the right that will navigate you to the next page.

Once the Sign Up form has loaded, you will need to enter your email address and full name on the right and and then you can click the Get Button.

Step 2: Getting your ESP API key

Once you have succesfully registered you will be navigated to your subscription page where you will find a section with a label "License key".

Click on the "Copy" button to get your API key ready for the next step.

Optionally you will also recieve an email with your API key which you can copy.

Step 3: Saving ESP API key in the settings tab

With the copied API key, come back to the ESP Loadshedding extension and navigate to the settings Tab by clicking on the Gear icon in the top right.

One on the settings page paste your API key in the "API Key" input field and click on the Save button.

Congratulations, You are all set :)

Very Important

You are limited to only one API key. So make sure you save it.

If you attempt to register again for a new API key, your account will be blocked for 24 hours.

Only after 24 hours will your account be reset where you can try again with a single API key.

Let's embark on this journey together and unlock the full potential of the Eskom Se Push extension!

Step 1: Sign up for a ESP api key

Go to the Eskom Se Push Gumroad Dashboard with the link below to sign up for a ESP API Key.

Copy Gumroad API Link
https://eskomsepush.gumroad.com/l/api

Once on the page click on the Subscribe button on the right that will navigate you to the next page.

Once the Sign Up form has loaded, enter your email address and click the Get Button. Once you are done with that you will recieve a email from "customers.gumroad.com" that wil contain your API key.

Step 2: Getting ESP api key from email

Once you have recieved your sign up email from "customers.gumroad.com", copy your API key.

Step 3: Saving ESP api key in settings tab

With the copied API key paste it in the "API Key" input field and click on the Save button.

Congratulations, You are all set
